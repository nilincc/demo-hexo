git部署代码笔记

作者: 36K 日期: March 1, 2025  11:26:25

这里记录一下，在本地把代码部署到GitHub的代码，方便以后的查询。

git上传代码

  • 1 git status
  • 2 git add –all 添加所有更改的文件
  • 3 git commit -m “你要commit的内容” 对这些文件进行 commit
  • 4 git push 最后同步到GitHub云端仓库
